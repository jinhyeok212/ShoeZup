# REQUIREMENTS.md

# 슈줍(ShoeZup) 요구사항 정의서

---

## 프로젝트 개요

본 문서는 스니커즈 리셀 가격 분석 서비스 “슈줍(ShoeZup)”의 MVP(Minimum Viable Product) 개발을 위한 기능적·비기능적 요구사항을 정의한다.

MVP 기준으로 작성되며, 이후 확장 기능은 별도 Phase 2로 관리한다.

---

# 1. 프로젝트 개요

## 1.1 프로젝트명

# 슈줍 (ShoeZup)

AI 기반 스니커즈 리셀 가격 분석 플랫폼

---

## 1.2 프로젝트 목적

스니커즈 리셀 거래 데이터를 분석하여 사용자가 현재 가격 상태와 가격 흐름을 객관적으로 판단할 수 있도록 지원하는 것을 목적으로 한다.

사용자는 상품 검색 후 다음 정보를 확인할 수 있다.

- 가격 추이
- 평균 대비 현재 가격 위치
- 사이즈별 프리미엄
- 브랜드별 가격 흐름
- 거래량 기반 가격 분석

---

# 2. 시스템 구성

| 구분 | 기술 |
| --- | --- |
| Frontend | React + JavaScript + Tailwind CSS + Axios + Recharts |
| Backend | Python + FastAPI + SQLAlchemy + Pydantic |
| AI/Data Server | Pandas + NumPy |
| Database | PostgreSQL |
| Data Analysis | Pandas + NumPy |
| Visualization | Recharts |
| Crawling | Python + BeautifulSoup + Selenium |
| Deployment | Vercel + Render 또는 Docker 기반 배포 |

---

## 기술 스택 설명

- React: 검색 화면 및 상품 상세 UI 구현
- Tailwind CSS: 빠른 UI 제작 및 반응형 디자인 구성
- Axios: Frontend와 FastAPI 서버 간 HTTP 통신
- Recharts: 가격 추이 및 분석 그래프 시각화
- FastAPI: 데이터 분석 및 API 서버 구현
- SQLAlchemy: PostgreSQL ORM 관리
- Pydantic: API 요청/응답 데이터 검증
- PostgreSQL: 상품 및 거래 데이터 저장
- Pandas/NumPy: 데이터 분석 및 통계 계산
- BeautifulSoup/Selenium: 리셀 데이터 수집 및 크롤링

# 5. 기능 요구사항 (Functional Requirements)

## FR-01 상품 검색 기능

### 설명

사용자는 상품명을 검색할 수 있어야 한다.

### 입력

```text
상품명 키워드
예: Jordan, Yeezy, Dunk
```

### 처리

- 상품명 포함 검색 수행
- 검색 결과 리스트 반환

### 출력

| 항목 |
| --- |
| 상품명 |
| 브랜드 |
| 최근 평균가 |
| 최근 거래가 |
| 최근 변동률 |

### 우선순위

높음

---

## FR-02 상품 상세 조회 기능

### 설명

사용자는 특정 상품의 상세 정보를 조회할 수 있어야 한다.

### 출력 정보

| 항목 |
| --- |
| 상품명 |
| 브랜드 |
| 출시가 |
| 최근 거래가 |
| 평균 거래가 |
| 최저 거래가 |
| 최고 거래가 |

### 우선순위

높음

---

## FR-03 가격 추이 그래프 기능

### 설명

사용자는 날짜별 리셀 가격 흐름을 그래프로 확인할 수 있어야 한다.

### 그래프 구성

| 축 | 내용 |
| --- | --- |
| X축 | 거래 날짜 |
| Y축 | 거래 가격 |

### 추가 지표

- 최근 7일 평균
- 최근 30일 평균
- 가격 변동률

### 우선순위

높음

---

## FR-04 현재 가격 위치 분석 기능

### 설명

현재 가격이 최근 평균 대비 어느 위치인지 계산해야 한다.

### 계산 방식

```text
(현재가격 - 최근평균가격) / 최근평균가격 × 100
```

### 출력 예시

```text
현재 가격은 최근 30일 평균 대비 6% 낮은 상태입니다.
```

### 우선순위

높음

---

## 이미지 기반 스니커즈 분석 기능

사용자가 캡처 이미지 또는 직접 촬영한 신발 이미지를 업로드하면 AI가 해당 스니커즈를 분석하고 관련 정보를 제공하는 기능

### 사용자 흐름

```text
신발 이미지 업로드
        ↓
AI 이미지 분석
        ↓
신발 모델 식별
        ↓
DB 내 상품 매칭
        ↓
가격 정보 및 분석 결과 출력
```

### 적용 예정 기술

| 구분 | 기술 |
| --- | --- |
| 이미지 분석 | CNN 기반 이미지 분류 |
| 객체 탐지 | YOLO |
| 특징 추출 | ResNet / EfficientNet |
| 유사 이미지 검색 | CLIP / Vector Embedding |
| 백엔드 연동 | FastAPI |
| 데이터 저장 | PostgreSQL + Vector DB(확장 가능) |
